## Description

Yet another open sourced Minecraft PvP Client!

-----

## Features

  - **Spotify Integration:** Real-time "Now Playing" HUD with album art, smooth scrolling titles, and animated progress bars
  - **Advanced HUD System:** Fully customizable modules including FPS, Spotify, and so much more with smooth rounded-rect rendering.
  - **Optimized Performance:** Lightweight core optimized to maintain high frame rates even with multiple active HUD modules.

-----

## Installation

1.  **Developer Setup:** Clone the repository and import it as an Eclipse/IntelliJ project. Ensure you have the 1.8.8 MCP environment configured.
2.  **JAR Export:** - Ensure `pack.png` is in your root resources folder to avoid the `ResourcePackListEntryDefault` crash.
      - Export the project as a Runnable JAR.
3.  **Launch:** Run the JAR via your preferred launcher (standard Minecraft Launcher or MultiMC).

-----

## Usage

  - **Click GUI:** Press `RSHIFT` (default) to open the module configuration menu.
  - **Spotify Module:** Enable the Spotify module in the HUD category. Ensure your `SpotifyAPI` is authenticated to fetch track data.
